# kasax_child（kasax Child Theme）

WordPressテーマ `kasax`（Underscoresベース）の機能を拡張・カスタマイズするための子テーマです。

## 概要

このテーマは、親テーマ `kasax` のデザインや機能を維持しつつ、独自のカスタマイズを加えるために作成されました。
子テーマを利用することで、親テーマのアップデートの影響を受けずに安全に機能を追加できます。

## 主なカスタマイズ内容

(※ここには、あなたが実際に追加・変更したことを箇条書きで書くとポートフォリオとして評価が高まります)

- **デザインの変更**: `style.css` によるカラーパレットやフォントのカスタマイズ。
- **機能の追加**: `functions.php` を使用した独自フックの追加。
- **テンプレートのオーバーライド**: `header.php` や `footer.php` の構造変更。

## インストール方法

1. 親テーマ（`kasax`）が `wp-content/themes/` ディレクトリにインストールされていることを確認してください。
2. この子テーマフォルダ（`kasax_child`）を `wp-content/themes/` にアップロードします。
3. WordPress管理画面の [外観] > [テーマ] から「kasax Child」を有効化します。
